# Changelog

## [1.0.0] - 2024-12-XX

### 첫 공개 릴리즈 🎉

#### 핵심 기능
- 병음/한글 뜻 검색으로 단어 자동 저장

- 획순 학습 (자동/한획씩/반복 애니메이션)

- 학습 통계
- 설정 (글씨 크기, 표시 옵션)
